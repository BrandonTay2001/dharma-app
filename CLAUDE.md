# Dharma – Spirituality App

A spirituality app focused on Buddhism and Hinduism, featuring scripture reading, spiritual chat, daily verses, guided meditation, mantras, and gratitude journaling.

## Project Structure

```
dharma-app/
├── dharma-frontend/     # iOS frontend (SwiftUI)
├── dharma-backend/      # Backend API (TBD)
└── project-guidelines/  # Design screens & design system docs
```

## Tech Stack

- **Frontend:** SwiftUI (iOS), Xcode project at `dharma/dharma.xcodeproj`
- **Backend:** To be placed in `dharma-backend/` (stack TBD)

## Core Features

1. **Scripture Reading** – Bhagavad Gita and Dhammapada (more sources to be added)
2. **Spiritual Chat** – AI guidance focused on Indian-style religions, hardcoded 5 messages/day limit
3. **Daily Verse** – Hindu and Buddhist verses, can be added to home screen
4. **Daily Tasks** – Daily verse, guided meditation, daily mantra, gratitude journaling

## Design System

- **Aesthetic:** Pure white foundation with saffron (#FF7722) accents
- **Typography:** Serif (Georgia) for scripture, system sans-serif for UI
- **Refer to:** `project-guidelines/design.md` and `project-guidelines/designs/sattva_clarity/DESIGN.md`
