# Design System: Dharma Spirituality App

**Visual Identity:** Minimalism, Purity, Spiritual Clarity
**Core Aesthetic:** Pure White Foundation with Saffron Accents

---

## 1. Visual Language

### 1.1 Color Palette
*   **Primary Background:** `#FFFFFF` (Solid Pure White)
    *   Used for all screen backgrounds, cards, and modal surfaces to create a sense of vastness and peace.
*   **Primary Accent:** `#FF7722` (Saffron)
    *   Used for: Call-to-action buttons, progress indicators, interactive icons, active navigation states, and highlighting key spiritual markers.
*   **Typography (Primary):** `#1A1A1A` (Near Black)
    *   Used for body text, headings, and labels to ensure maximum legibility against white.
*   **Secondary Text:** `#666666` (Medium Grey)
    *   Used for metadata, time stamps, and secondary descriptions.

### 1.2 Typography
*   **Content (Sacred Texts/Verses):** Elegant Serif (e.g., Playfair Display, Lora, Noto Serif).
    *   *Purpose:* To evoke a sense of tradition, reverence, and timeless wisdom.
*   **UI/Navigation:** Clean Sans-Serif (e.g., Inter, Roboto, Manrope).
    *   *Purpose:* To maintain modern usability, clarity, and structural balance.

---

## 2. Core Components & Patterns

### 2.1 The "Dharma" Row (Dashboard)
*   **Layout:** Full-width horizontal cards spanning the screen.
*   **Visuals:** Pure white background, subtle border or soft shadow for separation.
*   **Content:** Left-aligned icon/image and right-aligned status (e.g., 'DONE' badge in saffron).

### 2.2 Navigation (Bottom Bar)
*   **Items:** Chat, Today (Home), Scriptures, Learn, Settings.
*   **State:** Active item highlighted in Saffron (#FF7722) with a subtle background shift or icon fill.

### 2.3 Progress Indicators
*   **Style:** Thin, clean linear tracks or circular counters.
*   **Color:** Saffron (#FF7722) against a light grey or white track.

### 2.4 Buttons & Interactive Elements
*   **Primary CTA:** Solid Saffron background with White text.
*   **Secondary/Ghost:** Saffron border and text on White background.
*   **Roundness:** Medium-soft (8px - 12px) to maintain a friendly, accessible feel.

---

## 3. Screen Archetypes

### 3.1 Dashboard (Home)
*   Top section: Streak counter (Saffron flame) and Progress bar ("Your Path Today").
*   Middle section: Stacked horizontal task rows (Daily Verse, Meditation, Mantra).

### 3.2 Reading View (Scriptures)
*   Centered serif typography for the text.
*   Saffron-colored verse numbers or section headers.
*   Bottom persistent bar for navigation, audio controls, and text settings.

### 3.3 Chat Interface
*   Pure white background.
*   Saffron speech bubbles for AI/Guru responses.
*   White/Grey bubbles for User input.
*   Clear daily message limit indicator at the top.

### 3.4 Meditation Interface
*   Minimalist center focus (e.g., Lotus icon or breathing guide).
*   Saffron accents for timer and "Done" controls.
*   Extreme use of whitespace to reduce cognitive load.

---

## 4. Design Principles
1.  **Breathe:** Use generous whitespace. Never crowd the content.
2.  **Guide:** Use Saffron purposefully to lead the user to the next spiritual action.
3.  **Honor:** Treat scripture text with the highest level of legibility and respect.
4.  **Simplify:** Remove all non-essential decorative elements to maintain focus on the practice.
